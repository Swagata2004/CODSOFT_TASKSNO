A personal digital portfolio designed to showcase professional skills, projects, and achievements in an elegant and modern layout.

🏠 Home – Hero section with professional introduction and tagline

3. Advanced Calculator
Folder: WebDev-L1-Calculator

A feature-rich calculator web application with multiple themes, memory functions, and full arithmetic capabilities.

Features & Button Functionality:

| Category | Buttons | Functionality | |----------|---------|---------------| | Digits | 0-9 | Input numerical values | | Basic Operations | + | Addition | | | - | Subtraction | | | * | Multiplication | | | / | Division | | Memory Functions | MRC | Memory Recall/Clear (press once to recall, twice to clear) | | | M+ | Memory Add (adds current display value to memory) | | | M- | Memory Subtract (subtracts current display value from memory) | | Controls | ON/OFF | Toggle calculator power on/off | | | C | Clear all input and reset calculator | | | CE | Clear last entry | | | = | Calculate and display result | | | . | Decimal point | | | ± | Toggle positive/negative sign | | | % | Percentage calculation |

Themes Available:

🌙 Dark Blue – Professional dark blue theme for low-light environments
⚫ Dark Black – High-contrast dark mode for OLED displays
⚪ White – Clean light theme for daytime use
Display: Large, clear display area showing input expressions and results in real-time.

Tech Stack: HTML5, CSS3, JavaScript (Vanilla)
